# Runcord

## ⭐️ 프로젝트 소개
Runcord는 러닝 도중에 찍고 싶은 사진을 찍고
러닝 기록을 저장해, 그 기록과 찍은 사진을 함께 보여주는 앱입니다.

## ⚙️ 개발 환경 및 기술 스택

|목적|사용 라이브러리 or 기술|
|------|---|
|최소 버전|15.0|
|비동기 작업|RxSwift|
|로컬 저장소|Core Data|
|화면 프레임워크|UIKit|
|지도|MKMapView|
|사용자 위치| CLLocationManager|
|디자인패턴| coordinator, mvvm, mvc|
|아키텍처| 클린 아키텍처|
